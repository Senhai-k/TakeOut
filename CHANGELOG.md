# Changelog

## v0.1.0

首个可运行版本，包含用户端微信小程序、商家管理后台和 Spring Boot 后端服务。

### 已包含

- 用户端点餐流程：浏览、购物车、地址、下单、模拟支付、订单跟踪、确认完成和评价。
- 商家管理流程：登录、经营概览、订单处理、用户评价查看、分类管理、菜品管理和图片上传。
- 后端能力：统一响应、参数校验、异常处理、JWT 认证、JPA 持久化和 H2 文件库。
- 数据处理：后端优先，本地 fallback，统一保留订单支付方式、评价等本地补充字段。
- 工程检查：GitHub Actions 覆盖后端测试、管理后台构建和小程序 JS 语法检查。

### 当前限制

- 支付为模拟支付，未接入真实微信支付。
- 用户端为默认登录，未接入真实微信授权登录。
- 默认使用 H2 本地文件库，生产环境需要切换 MySQL 或其他数据库。
