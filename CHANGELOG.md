# Changelog

## v0.1.0

首个可运行版本。

### 已包含

- 用户端微信小程序：商家浏览、菜品搜索、规格选择、购物车、地址管理、确认订单、订单跟踪。
- 商家管理后台：登录、经营概览、订单处理、分类管理、菜品管理、图片上传。
- Spring Boot 后端：REST API、统一响应、参数校验、异常处理、JWT 认证、JPA 持久化。
- 数据存储：默认 H2 文件库，提供 MySQL profile 和初始化脚本参考。
- 订单流程：下单、模拟支付、商家接单/拒单、制作、配送、用户确认完成。
- 订单评价：已完成订单支持本地评价，订单详情展示评价结果。
- 商家收藏：小程序支持收藏商家，并在个人中心展示收藏数量。
- 支付方式：个人中心支持本地切换模拟支付方式。
- 个人中心：后端提供用户订单数、累计消费和奖励积分统计接口，小程序优先读取接口数据。
- 上传限制：图片类型和大小校验，默认支持 jpg、jpeg、png、webp、gif，最大 2MB。
- 自动化检查：GitHub Actions 运行后端测试、管理后台构建和小程序 JS 语法检查。

### 验证

- `mvn test`：26 个测试通过。
- `npm run build`：通过。
- 小程序 JS 语法检查：通过。
- GitHub Actions CI：通过。

### 当前限制

- 支付为模拟支付，未接入真实微信支付。
- 用户端为默认登录，未接入真实微信授权登录。
- 后台具备 JWT 认证，但还不是完整多角色权限系统。
- 默认使用 H2 本地文件库，生产环境需要切换 MySQL 或其他数据库。
