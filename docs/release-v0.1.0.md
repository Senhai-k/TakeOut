# v0.1.0 版本说明

`v0.1.0` 是 TakeOut 的首个可运行版本，包含用户端微信小程序、商家管理后台和 Spring Boot 后端服务。

## 主要功能

- 用户端：浏览商家和菜品、购物车、地址管理、提交订单、模拟支付、订单状态跟踪。
- 商家收藏：支持本地收藏商家并从个人中心快速进入。
- 订单评价：已完成订单支持本地评价和详情展示。
- 商家端：经营概览、订单处理、分类管理、菜品管理、图片上传。
- 后端服务：统一接口响应、参数校验、异常处理、认证保护、H2 文件库持久化。
- 工程配置：GitHub Actions CI、H2/dev/mysql profile、上传文件校验。

## 运行入口

- 后端：`server/`
- 管理后台：`admin-web/`
- 微信小程序：`miniapp/`

详细步骤见 [README](../README.md) 和 [运行检查清单](checklist.md)。

## 验证结果

- 后端测试：`mvn test`，26 个测试通过。
- 管理后台构建：`npm run build` 通过。
- 小程序 JS 语法检查通过。
- GitHub Actions CI 通过。

## 已知限制

- 支付使用模拟接口。
- 用户端暂未接入真实微信登录。
- 后台权限粒度仍可继续细化。
- 生产环境建议改用 MySQL，并增加数据库迁移工具。
