# v0.1.0 版本说明

`v0.1.0` 是 TakeOut 的首个可运行版本，包含用户端微信小程序、商家管理后台和 Spring Boot 后端服务。

## 主要功能

- 用户端：浏览、购物车、地址、下单、模拟支付、订单跟踪、确认完成和评价。
- 商家端：经营概览、订单处理、分类管理、菜品管理和图片上传。
- 后端服务：统一接口响应、参数校验、异常处理、认证保护和 H2 文件库持久化。
- 数据策略：后端优先，本地 fallback，并保留支付方式、评价等订单本地元数据。

## 运行入口

- 后端：`server/`
- 管理后台：`admin-web/`
- 微信小程序：`miniapp/`

详细步骤见 [README](../README.md) 和 [运行检查清单](checklist.md)。

## 已知限制

- 支付使用模拟接口。
- 用户端暂未接入真实微信登录。
- 生产环境建议改用 MySQL，并增加数据库迁移工具。
