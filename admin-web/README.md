# admin-web

商家端和平台管理后台目录。 

当前已提供一个最小可运行的 Vue 3 + Vite 后台骨架，可直接读取后端 `/api/merchant/*` 接口。

建议模块：

- `src/views/`：页面
- `src/components/`：公共组件
- `src/api/`：接口封装
- `src/router/`：路由
- `src/stores/`：状态管理
- `src/utils/`：工具函数
- `public/`：静态资源

## 本地启动

```bash
npm install
npm run dev
```

默认访问地址：

```text
http://localhost:5173
```
